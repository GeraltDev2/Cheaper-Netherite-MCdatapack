<h1 align="center">Minecraft Datapack: <br> Cheaper Netherite</h1>

# Features

- Craft netherite ingots with less netherite scrap and gold ingots.
- Craft netherite upgrade smithing templates with two less diamonds.
- Rarely find netherite scrap in nether fortress chests.

# Issues

There are no known issues. Though this is the first datapack of mine that i have turned into a mod. If anything is not working propertly feel free to open an issue page.
